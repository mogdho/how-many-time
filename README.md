# How many times I have done it

A premium, private, and secure single-action logging web app.

## Features
- **Midnight Black UI**: Distraction-free materialistic design.
- **Secure PIN Lock**: SHA-256 hashed authentication via Web Crypto API.
- **Panic Reset**: Automatic data wipe after 10 failed attempts.
- **Visual Storytelling**: Weekly comparison charts (Last Week vs This Week).
- **PWA Support**: Installable to Home Screen, runs offline.
- **100% Private**: Your data never leaves your device (IndexedDB).

## Tech Stack
- React
- Vite
- Framer Motion
- Chart.js
- IndexedDB

## Local Development
```bash
npm install
npm run dev
```
